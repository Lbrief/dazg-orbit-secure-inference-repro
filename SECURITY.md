# Security policy

This repository is a research reproducibility artifact. The included execution lanes use reveal/re-share diagnostic adapters and explicitly set `security_claim=0`.

Report implementation vulnerabilities or accidental credential/data exposure privately to the repository owner. Do not file a public issue containing secrets, private datasets, or exploit details.

A correctness result in this repository must not be interpreted as a proof of end-to-end confidentiality. See `docs/SECURITY_BOUNDARY.md`.
