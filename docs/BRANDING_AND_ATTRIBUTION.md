# Branding and attribution

Active first-party code, build targets, macros, environment variables, and
runtime diagnostics use **DAZG-Orbit** naming. Historical identifiers such as
SEC-PPDL and PrivCirNet appear only in provenance records where removing them
would make attribution inaccurate.

Third-party source is not rebranded. Its copyright notices, namespaces, and
license text remain governed by the corresponding upstream project.
